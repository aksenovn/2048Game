# Игра 2048 на Python (Console & GUI)

Популярная игра-головоломка **2048**, реализованная на языке Python. Проект поддерживает два режима работы:
1. **Графический интерфейс (GUI)** на базе библиотеки `tkinter`.
2. **Консольная версия** для игры прямо в терминале.

---

## 🎮 Как играть

Цель игры — объединять плитки с одинаковыми числами, чтобы получить плитку с номиналом **2048**.

* При каждом ходе все плитки сдвигаются в выбранном направлении.
* Плитки с одинаковыми значениями при столкновении объединяются в одну с суммой их значений (например, 2 + 2 = 4, 4 + 4 = 8).
* После каждого хода на случайном пустом месте появляется новая плитка номиналом 2 (с вероятностью 90%) или 4 (с вероятностью 10%).
* Игра заканчивается, если на поле нет свободных ячеек и невозможно сделать ни одного хода для объединения плиток.

---

## 🛠️ Требования и установка

Для запуска игры вам понадобится установленный интерпретатор **Python 3.x**. 

1. Склонируйте репозиторий или скачайте файлы проекта.
2. Никаких дополнительных внешних библиотек устанавливать не нужно — используются только встроенные модули Python (`tkinter`, `random`, `os`).

---

## 🚀 Запуск игры

### 🖥️ Графическая версия (GUI)

Запустите файл `gui_2048.py` через терминал:
```bash
python gui_2048.py
```
* **Управление**: Используйте **стрелки клавиатуры** (Вверх, Вниз, Влево, Вправо) для сдвига плиток.
* **Перезапуск**: Если игра окончена, нажмите клавишу **`R`** на клавиатуре для старта новой игры.

### 📟 Консольная версия

Запустите файл `game_2048.py` через терминал:
```bash
python game_2048.py
```
* **Управление**: Вводите символы и нажимайте `Enter`:
  * `w` — Вверх
  * `s` — Вниз
  * `a` — Влево
  * `d` — Вправо

---

## 📁 Структура проекта

* `game_2048.py` — Логика игры (правила, сетка, слияние плиток, подсчет очков). Также содержит точку входа для запуска консольной версии.
* `gui_2048.py` — Графическая оболочка игры, обрабатывающая нажатия клавиш и отрисовку сетки с помощью `tkinter`.
* `__init__.py` — Файл инициализации пакета.
* `.gitignore` — Настройки исключения временных файлов Python (`__pycache__`) из системы контроля версий Git.
